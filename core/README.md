# Core

Shared utilities, interfaces, and helpers for the agent system.

## Contents

- **Utilities** - Common helper functions
- **Interfaces** - Shared type definitions
- **Validators** - Validation logic
- **Configuration** - Default settings and constants
- **Templates** - Base code templates

### Internal Starter Baseline

For Angular portal initialization, the baseline templates are maintained here:

- `core/templates/angular-portal-starter/package.template.json`
- `core/templates/angular-portal-starter/tsconfig.template.json`
- `core/templates/angular-portal-starter/structure.txt`

These files are the only source of truth for starter generation defaults.

## Purpose

The core module provides the foundation for all skills and agents to build upon, ensuring consistency and reducing duplication across the codebase.
